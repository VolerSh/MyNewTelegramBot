Детализация: Модуль Telegram-бота

← Назад к общей структуре

→ На входе

Модуль получает объект `message` от API Telegram, когда пользователь отправляет команду (например, /search) или любое текстовое сообщение.

↓ Что делает

1. Инициализация (bot.py): Создает и настраивает экземпляр бота, используя токен из config.py. Запускает бесконечный процесс опроса серверов Telegram.

2. Маршрутизация (handlers.py): Принимает входящее сообщение и, в зависимости от его содержания (команда или текст), направляет его в соответствующую функцию-обработчик.

3. Обработка: Выполняет логику, заложенную в обработчике. Для команды /search это означает вызов функции run_spider() из модуля парсинга.

← На выходе

Модуль отправляет данные во внешние системы:

  • Пользователю в Telegram: Ответные сообщения (например, "Начинаю поиск...").
  • Модулю `scraping`: Вызов функции `run_spider()` для запуска процесса парсинга.
